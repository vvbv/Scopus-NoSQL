# Scopus NoSQL*
_<strong><small>* Repositorio base para la construcción de la imagen.</small></strong>_  
<hr>
El siguiente repositorio contiene todo lo necesario para la construcción de una imagen de docker, la cual va a contener una base de datos almacenada en <strong>MongoDB</strong> con los metadatos formateados de inicialmente 3000 artículos que contienen información referente al cancer, publicados en <strong>Scopus.com</strong> entre los años 2017 y 2019.
<hr>

###  Descarga y ejecución [ <a href="https://hub.docker.com/r/vvbv/nosqlscopus">dockerhub</a> ]:

```bash
docker run -it vvbv/nosqlscopus:v3
```
